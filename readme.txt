This is a booking form plugin for sites using musubiapp.com's services.
 
== Description ==
 
Set-up your service schedules at musubiapp.com and use this simple plugin to show booking forms on your website and allow users to book your services.

You will require a Musubi Account and to set-up your schedules with Musubi Bookings before using this plugin.
 
== Installation ==
 
1. Create an account at musubiapp.com.
2. Set up your service schedules at admin.musubiapp.com. Make sure to remember the id code the system provides you.
3. Download the plugin from your WordPress' Admin Panel. Alternatively, download the plugin from admin.musubiapp.com and upload the `musubibookings` folder to the `/wp-content/plugins/` directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Use a widget or shortcode with the code you got during step 2.
6. Enjoy!
